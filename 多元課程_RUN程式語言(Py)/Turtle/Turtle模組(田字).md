# Turtle 模組(田字)
```py
import turtle
turtle.clear()
for i in range(0,4,1):
  for i in range(0,4,1):
    turtle.forward(50)
    turtle.right(90)
  turtle.forward(100)
  turtle.right(90)
turtle.done()
```
